# naive_chain.cr

Implementation of [lhartikk/naivechain](https://github.com/lhartikk/naivechain.git) in Crystal.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  naive_chain.cr:
    github: TobiasGSmollett/naive_chain.cr
```

## Usage

```crystal
require "naive_chain"
```
